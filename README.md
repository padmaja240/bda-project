# Smart Customer Purchase Analytics & Recommendation System

This repository contains the complete **Big Data Analytics (BDA) project evaluation package** for GitHub submission.

## Project Overview
The project analyzes customer purchase behaviour using historical transaction data. It applies data cleaning, exploratory data analysis, customer segmentation, and a simple recommendation approach to generate business insights. The dashboard presents results through charts, summary cards, and tables.

## Execution Steps
1. Open terminal in:
   `03_Implementation/Flask_Dashboard`

2. Install dependencies:
   ```bash
   pip install -r ../../requirements.txt
   ```

3. Run the project:
   ```bash
   python app.py
   ```

4. Open in browser:
   ```bash
   http://127.0.0.1:5000
   ```
